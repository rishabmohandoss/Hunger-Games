# Agent Instructions for Hunger-Games

## Changelog Requirement

**Every change made to this repository — by any agent or contributor — must be recorded in `CHANGELOG.md` before committing.**

Each entry must include:
- The date the change was made (`YYYY-MM-DD`)
- The agent or author name
- The file and line number affected
- What was changed
- Why it was changed (the motivation, not just a description of the diff)

Follow the format already established in `CHANGELOG.md`. Add new entries at the top, below the header.

Failure to update `CHANGELOG.md` before committing is a mistake — always include it in the same commit as the code change.
